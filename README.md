# Grupo - eso-no-vale
Proyecto de python de videojuego buscaminas y hundir la flota.

Este repositorio contiene una aplicación de escritorio desarrollada en **Python** que ofrece una interfaz sencilla para disfrutar de dos juegos de mesa: el **Buscaminas** y **Hundir la Flota**.

---

## Juegos

#### Buscaminas
El objetivo es despejar un tablero de cuadrados que ocultan minas sin detonar ninguna, usando pistas sobre el número de minas vecinas en cada campo, pudiendo elegir entre diferentes dificultades.

#### Hundir la Flota
El clásico juego de estrategia naval donde deberás adivinar la posición de los barcos del oponente para hundirlos antes de que ellos destruyan tu flota, en este juego, jugarás contra la IA.
